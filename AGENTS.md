## Rules
- Do not use React, Vue, Node.js, FastAPI, or external libraries.
- Keep the site deployable on GitHub Pages.
- Preserve semantic HTML structure.
- Do not rewrite Japanese profile text unless asked.
- Keep the code understandable for a beginner.
- Explain changes in Japanese.
- Before editing, explain the plan.
- After editing, summarize changed files and test points.
- Make one small change at a time.
- Do not change multiple features at once unless asked.
