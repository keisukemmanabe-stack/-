About Me

私は現在、大学4年生として、医療系・ICT系の技術を用いて課題解決を目指す研究室に所属しています。

特に、医療現場で扱われる情報や画像、データをどのように活用できるかに関心があります。 AIやデータ分析によって得られた結果を、Webアプリケーションを通じて分かりやすく表示し、 利用者にとって使いやすい形にすることを学んでいきたいです。

私がWebアプリケーション開発を学びたい理由は、AIやデータ分析の結果を、実際に人が使える形にする力が重要だと感じているからです。 どれだけ良い分析結果やAIモデルがあっても、利用者にとって見やすく、操作しやすく、理解しやすい形で提供できなければ、実用にはつながりにくいと考えています。

そのため、今回の自己紹介ページ制作では、HTMLで意味のある構造を作り、CSSで見やすく整え、JavaScriptでユーザー操作に応じた変化を加えるという、 フロントエンド開発の基本的な流れを意識しています。

Interests

私の興味分野は、医療ICT、AI、データ分析、Webアプリケーション開発です。

医療ICT

医療ICTに関心があります。医療現場では、電子カルテ、検査データ、画像データなど 多くの情報が扱われています。これらの情報を分かりやすく整理し、必要な人が使いやすい形で 活用できる仕組みを作ることで、医療現場の課題解決に貢献できると考えています。

AI

AIについては、画像分析や疾患予測などへの応用に関心があります。 一方で、AIを実際に活用するためには、精度だけでなく、説明可能性、使いやすさ、 安全性も重要だと感じています。

データ分析

データ分析では、データに基づいて状況を理解し、意思決定を支援することに関心があります。 医療・工学・金融など、さまざまな分野の課題解決に応用できる技術を身につけたいです。

Webアプリケーション開発

Webアプリケーション開発は、AIやデータ分析の結果を、実際に人が使える形にするための 重要な技術だと考えています。HTML / CSS / JavaScriptを学び、見やすく、操作しやすく、 目的に合った画面を作れるようになることを目指しています。

Learning

現在は、HTML / CSS / JavaScriptを中心に、Webフロントエンド開発の基礎を学んでいます。

HTML / CSS / JavaScript

HTMLではページの構造を作り、CSSでは見た目やレイアウトを整え、 JavaScriptではユーザーの操作に応じて画面を変化させる方法を学んでいます。

Machine Learning

これまで機械学習では、分類モデル、ニューラルネットワーク、深層強化学習などを学んできました。 その中で、精度だけでなく、リアルタイム速度やモデルサイズなど、実際に使うための観点も 重要だと感じるようになりました。

Python × Web

将来的には、Pythonで行った分析やAIモデルを、Webアプリケーションと連携させる開発にも 挑戦したいです。
